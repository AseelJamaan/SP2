## Model Input Analysis

In the third task, our goal was to understand the structure and format of the data that will be used as input for the model.

To achieve this, we analyzed the research paper that we selected as the main reference for our project, as well as other related papers that discuss the integration of Reinforcement Learning with Adaptive Large Neighborhood Search (ALNS).

Each team member conducted an individual search to identify how previous studies represent the state, actions, and other variables used by the model. This helped us understand what type of information should be provided to the model so that it can effectively learn and improve the search process.

To organize our findings, we created a shared Google document where each team member records the results of their search and summarizes the key insights they discovered from the papers they reviewed.
