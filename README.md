## Project Overview

This project focuses on optimizing school transportation systems using advanced optimization and machine learning techniques. The main objective is to improve the efficiency of school bus routing while minimizing travel time and distance, and ensuring that operational constraints are satisfied.

In the first phase of the project (SP1), the team developed an optimized routing framework using clustering techniques and the Adaptive Large Neighborhood Search (ALNS) algorithm to generate efficient routes.

In the second phase (SP2), the project extends the previous work by integrating **Reinforcement Learning (RL)** with the **Adaptive Large Neighborhood Search (ALNS)** algorithm. The goal is to enhance the search process dynamically and improve the overall routing performance.
