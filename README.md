## Task 4: Dataset Processing and Clustering

In this task, each team member is responsible for processing assigned dataset files and preparing them for clustering.

Each member should download the dataset files available in the GitHub repository and organize them locally to ensure that each file is handled by the correct team member.

The required steps for each assigned file include:

- Rename the distance columns using the format:
  **distance_p1, distance_p2, ...**
- Rename the time columns using the format:
  **time_p1, time_p2, ...**
- Merge the time data with the corresponding distance file by copying the time columns into the distance dataset.
- Apply clustering on the processed dataset.
- Save the clustering results by adding a new column named:
  **Cluster_Id**
